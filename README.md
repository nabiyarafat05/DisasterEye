# DisasterEye

AI-based disaster image analyzer intended to inspect satellite or drone images, identify disaster type, assess severity, and generate intelligence reports with Google Gemini Vision API.

## Current Status

The project is still at an early skeleton stage:

- `main.py` contains a minimal CustomTkinter window with upload and analyze controls.
- `ui.py` is empty.
- `gemini_ai.py` is empty.
- `image_processor.py` is empty.
- `report_generator.py` is empty.
- `utils.py` is empty.
- `requirements.txt` is empty.

## Project Structure

- `assets/uploads/` is present for uploaded images.
- `assets/uploads/reports/` is present for generated reports.

## Notes

No analysis pipeline, API integration, or report generation logic has been implemented yet.
